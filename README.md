## Learning Platforms

- [Hack The Box](https://www.hackthebox.com) - Online platform to practice penetration testing skills with real-world machines.
- [TryHackMe](https://tryhackme.com) - Beginner-friendly, guided learning paths for cybersecurity.
- [PentesterLab](https://pentesterlab.com) - Hands-on web application security exercises.
- [PortSwigger Web Security Academy](https://portswigger.net/web-security) - Free, comprehensive web security training from the makers of Burp Suite.
- [VulnHub](https://www.vulnhub.com) - Download vulnerable VMs to practice locally.
- [OverTheWire](https://overthewire.org/wargames/) - Wargames to learn and practice security concepts.
- [PicoCTF](https://picoctf.org) - CTF platform geared toward beginners and students.
- [Root-Me](https://www.root-me.org) - Hacking and information security challenges.

## CTF (Capture The Flag)

- [CTFtime](https://ctftime.org) - Calendar and archive of CTF competitions worldwide.
- [CTF101](https://ctf101.org) - Introduction to common CTF categories and techniques.
- [pwndbg](https://github.com/pwndbg/pwndbg) - GDB plugin for exploit development and reverse engineering.
- [pwntools](https://github.com/Gallopsled/pwntools) - CTF framework and exploit development library.
- [angr](https://github.com/angr/angr) - Binary analysis platform for automated CTF challenges.
- [GEF](https://github.com/hugsy/gef) - GDB Enhanced Features for exploit developers.

## Penetration Testing

- [Metasploit Framework](https://www.metasploit.com) - The world's most widely used penetration testing framework.
- [Kali Linux](https://www.kali.org) - Debian-based Linux distribution for penetration testing.
- [Parrot OS](https://www.parrotsec.org) - Security-focused GNU/Linux distribution.
- [OWASP Testing Guide](https://owasp.org/www-project-web-security-testing-guide/) - Comprehensive guide for web application penetration testing.
- [PayloadsAllTheThings](https://github.com/swisskyrepo/PayloadsAllTheThings) - Useful payloads and bypass techniques for web applications.
- [HackTricks](https://book.hacktricks.xyz) - Hacking tricks, techniques, and cheatsheets.

## Web Application Security

- [OWASP Top 10](https://owasp.org/www-project-top-ten/) - The ten most critical web application security risks.
- [Burp Suite](https://portswigger.net/burp) - Leading toolkit for web security testing.
- [OWASP ZAP](https://www.zaproxy.org) - Free, open-source web application scanner.
- [SQLMap](https://sqlmap.org) - Automated SQL injection and database takeover tool.
- [XSStrike](https://github.com/s0md3v/XSStrike) - Advanced XSS detection and exploitation suite.
- [Nikto](https://github.com/sullo/nikto) - Open-source web server scanner.
- [Wfuzz](https://github.com/xmendez/wfuzz) - Web application fuzzer.

## Network Security

- [Nmap](https://nmap.org) - Network discovery and security auditing tool.
- [Wireshark](https://www.wireshark.org) - Network protocol analyzer.
- [Masscan](https://github.com/robertdavidgraham/masscan) - Fast TCP port scanner.
- [Netcat](https://netcat.sourceforge.net) - Networking utility for reading/writing network connections.
- [Scapy](https://scapy.net) - Python-based packet manipulation program.
- [Responder](https://github.com/lgandx/Responder) - LLMNR, NBT-NS and MDNS poisoner.
- [Impacket](https://github.com/fortra/impacket) - Python classes for working with network protocols.

## Reverse Engineering

- [Ghidra](https://ghidra-sre.org) - NSA's open-source software reverse engineering framework.
- [IDA Pro](https://hex-rays.com/ida-pro/) - Industry-standard disassembler and debugger.
- [Binary Ninja](https://binary.ninja) - Modern reverse engineering platform.
- [Radare2](https://rada.re/n/) - Open-source reverse engineering framework.
- [x64dbg](https://x64dbg.com) - Open-source debugger for Windows.
- [OllyDbg](http://www.ollydbg.de) - x86 debugger for Windows executables.
- [dnSpy](https://github.com/dnSpy/dnSpy) - .NET debugger and assembly editor.

## Exploit Development

- [ROPgadget](https://github.com/JonathanSalwan/ROPgadget) - Tool to search for ROP gadgets in binaries.
- [pwntools](https://github.com/Gallopsled/pwntools) - CTF framework for writing exploits.
- [checksec](https://github.com/slimm609/checksec.sh) - Script to check binary security properties.
- [one_gadget](https://github.com/david942j/one_gadget) - Tool for finding one-gadget RCE in libc.
- [LibcSearcher](https://github.com/lieanu/LibcSearcher) - Python library to search for libc functions.
- [ret2libc explained](https://ir0nstone.gitbook.io/notes) - Binary exploitation notes by ir0nstone.

## Malware Analysis

- [Any.run](https://any.run) - Interactive online malware sandbox.
- [VirusTotal](https://www.virustotal.com) - Analyze suspicious files, URLs, and IPs.
- [Cuckoo Sandbox](https://cuckoosandbox.org) - Automated malware analysis system.
- [FLARE VM](https://github.com/mandiant/flare-vm) - Windows-based security distribution for malware analysis.
- [REMnux](https://remnux.org) - Linux toolkit for reverse-engineering and analyzing malware.
- [pestudio](https://www.winitor.com) - Malware initial assessment tool for Windows executables.
- [YARA](https://virustotal.github.io/yara/) - Pattern matching tool for malware researchers.

## OSINT

- [Maltego](https://www.maltego.com) - Interactive data mining and link analysis tool.
- [Shodan](https://www.shodan.io) - Search engine for internet-connected devices.
- [theHarvester](https://github.com/laramies/theHarvester) - OSINT tool for gathering emails, subdomains, and more.
- [Recon-ng](https://github.com/lanmaster53/recon-ng) - Web reconnaissance framework.
- [SpiderFoot](https://www.spiderfoot.net) - Automated OSINT collection and analysis.
- [OSINT Framework](https://osintframework.com) - Collection of OSINT tools and resources.
- [Censys](https://censys.io) - Search engine for internet infrastructure.

## Cryptography

- [CyberChef](https://gchq.github.io/CyberChef/) - Web-based data analysis and encoding tool.
- [HashCat](https://hashcat.net/hashcat/) - Advanced CPU-based password recovery utility.
- [John the Ripper](https://www.openwall.com/john/) - Open-source password security auditing tool.
- [CrackStation](https://crackstation.net) - Online hash cracking using large lookup tables.
- [dCode](https://www.dcode.fr/en) - Tools for ciphers, codes, and cryptanalysis.
- [RsaCtfTool](https://github.com/RsaCtfTool/RsaCtfTool) - RSA attack tool for CTF challenges.

## Tools

### Reconnaissance
- [Amass](https://github.com/owasp-amass/amass) - In-depth attack surface mapping and asset discovery.
- [subfinder](https://github.com/projectdiscovery/subfinder) - Subdomain discovery tool.
- [httpx](https://github.com/projectdiscovery/httpx) - Fast and multi-purpose HTTP toolkit.
- [ffuf](https://github.com/ffuf/ffuf) - Fast web fuzzer.
- [feroxbuster](https://github.com/epi052/feroxbuster) - Fast, recursive content discovery tool.

### Post-Exploitation
- [LinPEAS](https://github.com/carlospolop/PEASS-ng) - Linux privilege escalation scripts.
- [WinPEAS](https://github.com/carlospolop/PEASS-ng) - Windows privilege escalation scripts.
- [BloodHound](https://github.com/BloodHoundAD/BloodHound) - Active Directory attack path visualization.
- [Mimikatz](https://github.com/gentilkiwi/mimikatz) - Windows credential extraction tool.
- [CrackMapExec](https://github.com/byt3bl33d3r/CrackMapExec) - Swiss army knife for Windows/AD environments.

### Wireless
- [Aircrack-ng](https://www.aircrack-ng.org) - Suite of tools for Wi-Fi network security assessment.
- [Wifite](https://github.com/derv82/wifite2) - Automated wireless auditing tool.
- [Kismet](https://www.kismetwireless.net) - Wireless network detector and sniffer.

## Courses & Certifications

- [OSCP (Offensive Security Certified Professional)](https://www.offsec.com/courses/pen-200/) - Industry-standard hands-on penetration testing certification.
- [eJPT (eLearnSecurity Junior Penetration Tester)](https://ine.com/learning/certifications/internal/elearnsecurity-junior-penetration-tester) - Entry-level penetration testing certification.
- [CEH (Certified Ethical Hacker)](https://www.eccouncil.org/train-certify/certified-ethical-hacker-ceh/) - EC-Council's ethical hacking certification.
- [PNPT (Practical Network Penetration Tester)](https://certifications.tcm-sec.com/pnpt/) - Practical, affordable pentest certification by TCM Security.
- [TCM Security Courses](https://academy.tcm-sec.com) - Affordable, high-quality hacking courses.
- [SANS Courses](https://www.sans.org) - Professional cybersecurity training and certifications (GPEN, GWAPT, etc.).
